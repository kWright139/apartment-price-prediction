# Apartment Price Prediction Project

CAP4770 Data Mining Project

## Overview
This project analyzes U.S. apartment rental data to explore pricing trends and build predictive models for rental prices.

## Main Notebook
The complete analysis, data cleaning, visualization, and modeling are contained in:

notebooks/apartment_price_analysis_full.ipynb

Download the dataset and place it in your working directory before running the notebook.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- SQLite (for database storage)

## How to Run
1. Download the dataset from Google Drive
2. Place the CSV file in your project directory
3. Open the notebook:
   notebooks/apartment_price_analysis_full.ipynb
4. Run all cells

## Team Responsibilities
- Member A: Problem Statement & Database Setup
- Member B: Data Cleaning & Exploratory Data Analysis
- Member C: Modeling & Evaluation

## Key Insights
- Most rental prices fall between $800–$2000
- Square footage has a moderate positive relationship with price
- Location significantly impacts rental pricing
